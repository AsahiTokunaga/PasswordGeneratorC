# Password Generator in C
# Used
1. clone repo
```
git clone https://github.com/AsahiTokunaga/PasswordGeneratorC.git
```
2. compile
```
make all
```
3. execute
```
./pwgenC [passwordLength] [charTypes(U/L/N/S)]
```

> [!TIP]
> charTypes can be selected more than once.
